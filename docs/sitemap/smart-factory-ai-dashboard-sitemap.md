## 3. Detailed Page Descriptions

### 3.1 Authentication

1. Login: allows managers, staff, and safety teams to access the system.
2. Forgot Password: lets users request a password reset.
3. Reset Password: allows users to create a new password.
4. Optional Role Selection: useful in demo mode to switch between different user roles.

### 3.2 Main Dashboard

1. KPI Overview: displays production output, active lines, alerts, workforce count, and warehouse capacity.
2. Active Alerts Summary: shows critical production and safety incidents.
3. Production Summary: gives a quick snapshot of line performance.
4. Warehouse Summary: shows stock status and placement issues.
5. Workforce Summary: shows attendance, shift coverage, and overtime status.
6. Quick Actions: links to common actions such as creating forms or reviewing alerts.

### 3.3 Warehouse Management

1. Warehouse Overview: general warehouse status and occupancy.
2. Item Location Tracking: locate goods by item code or storage zone.
3. Product Search: find products quickly.
4. Storage Recommendations: AI suggestions for better storage placement.
5. Inventory Status: view stock levels and low-stock warnings.
6. Goods Movement History: track item movement records.
7. Wrong Placement Alerts: identify misplaced goods.

### 3.4 Production Monitoring

1. Line Overview: list all production lines and statuses.
2. Line Detail: view metrics for a selected line.
3. Output Tracking: compare actual output against targets.
4. Downtime Monitoring: monitor machine stoppage and recovery.
5. Defect Monitoring: track defect frequency and impact.
6. Performance Comparison: compare line efficiency side by side.
7. Historical Performance: review past performance data.

### 3.5 AI Camera Monitoring

1. Camera Overview: list all active cameras.
2. Live Camera Feed: show simulated live streams.
3. Incident Detection Log: record abnormal events.
4. Restricted Zone Detection: detect entries into forbidden areas.
5. Obstacle Detection: identify objects blocking operations.
6. Alert Severity View: organize incidents by severity.

### 3.6 Safety Management

1. Safety Overview: summary of current safety conditions.
2. Factory Risk Map: map dangerous zones visually.
3. Active Safety Alerts: display open warnings and critical risks.
4. Incident Records: store historical safety incidents.
5. High-Risk Area Report: identify repeated risk zones.
6. Early Warning Suggestions: AI-generated preventive suggestions.

### 3.7 Productivity Analytics

1. Productivity Dashboard: high-level performance overview.
2. Line Productivity Analysis: detailed productivity metrics per line.
3. Trend Analysis: view productivity changes over time.
4. Peak Efficiency Time Analysis: find the most effective operating periods.
5. Defect Rate Analysis: analyze quality issues.
6. AI Improvement Suggestions: recommend optimization actions.

### 3.8 Workforce and Shift Planning

1. Workforce Overview: employee count and availability.
2. Shift Planning: build daily or weekly shift plans.
3. Overtime Suggestions: recommend overtime when needed.
4. Employee Allocation by Line: assign staff based on skills and demand.
5. Availability and Absence Tracking: manage leave and absences.
6. AI Scheduling Recommendations: suggest optimized staffing plans.

### 3.9 Forms and Workflow Automation

1. Forms Overview: central page for all internal forms.
2. Leave Request Form: submit leave requests.
3. Overtime Request Form: request overtime.
4. Machine Issue Report: report machine faults.
5. Warehouse Import Request: request stock intake.
6. Warehouse Export Request: request stock release.
7. Form Approval Status: track approval progress.

### 3.10 Notifications and Alerts

1. Notification Center: unified inbox for all alerts.
2. Production Alerts: notify users about line issues.
3. Safety Alerts: notify users about dangerous conditions.
4. Staffing Alerts: notify users about manpower shortages.
5. Approval Notifications: notify users about form decisions.

### 3.11 Reports and Statistics

1. Reports Overview: entry point for all reports.
2. Production Reports: summarize output and line performance.
3. Warehouse Reports: summarize stock and storage efficiency.
4. Safety Reports: summarize incidents and risk areas.
5. Workforce Reports: summarize attendance, overtime, and staffing.
6. Export Reports: export reports to PDF or Excel.

### 3.12 Settings and Administration

1. User Management: manage users and accounts.
2. Role and Permission Settings: assign access rights.
3. Factory Configuration: define plant areas and departments.
4. Line Configuration: configure production lines.
5. Alert Threshold Settings: configure warning rules.
6. AI Recommendation Rules: adjust AI-based suggestion logic.
7. Notification Preferences: manage alert channels and preferences.
